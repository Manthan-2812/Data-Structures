# Data Structures in C (Practicals)

This repository contains implementations of core Data Structures and Algorithms in C,
written as part of academic practice and practical exam preparation.

## Topics Covered
- Stack (Array & Linked List)
- Queue (Linear, Circular, Linked List)
- Linked Lists
- Searching Algorithms
- Sorting Algorithms
- Trees (BST)
- Hashing

## Note
Each program is written independently for clarity and practical exam use.
